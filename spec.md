# Aghori Tattoo Studio Website

## Overview
A single-page website for Aghori Tattoo Studio showcasing their services and portfolio with a dark, modern tattoo aesthetic.

## Design Theme
- Dark modern aesthetic using black, gray, and red color scheme
- Tattoo-inspired styling and typography
- Fully responsive design for mobile and desktop

## Sections

### Hero Section
- Studio name "Aghori Tattoo Studio" prominently displayed
- Striking tattoo-themed background image
- Call-to-action elements

### About Section
- Information about Aghori Tattoo Studio
- Studio location: Chhota Shimla
- Studio philosophy and services

### Gallery Section
- Grid layout showcasing sample tattoo designs
- Placeholder images for tattoo artwork
- Responsive image gallery

### Contact Section
- Phone number: 8091694910
- Contact form with fields:
  - Name (required)
  - Message (required)
- Form submission handling

## Technical Requirements
- Single-page application
- No user authentication required
- Contact form submissions stored in backend
- Static content (no dynamic content management needed)

## Backend Data Storage
- Contact form submissions (name, message, timestamp)

## Backend Operations
- Store contact form submissions
- Retrieve contact submissions for studio owner
